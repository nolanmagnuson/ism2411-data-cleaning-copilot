# ism2411-data-cleaning-copilot

This project cleans a messy sales dataset using Python and pandas.  
The goal is to fix inconsistent column names, whitespace issues,
missing values, and invalid numbers.

## How to Run

1. Make sure the raw CSV file is located at:
   `data/raw/sales_data_raw.csv`

2. Run the script from the project root folder:

3. The cleaned file will appear at:
   `data/processed/sales_data_clean.csv`
